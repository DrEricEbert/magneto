# UNITS

VARIABLE					UNITS				DESCRIPTION
sample_rate 				Hz					sample rate for the DAC and ADC
recordings_per_position		N/A 				number of times a pulse should be output and recorded
pulse_spacing				ms					time to wait between adjacent pulses
amplitude					V 					maximum amplitude
